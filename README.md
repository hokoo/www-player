# WWW Player

Мини-приложение для браузера с отдельными кнопками воспроизведения для каждого аудиофайла, индивидуальными уровнями громкости и перекрытием дорожек (overlay).

## Запуск

1. Убедитесь, что у вас установлен Node.js (v18+).
2. Поместите аудиофайлы (mp3, wav, ogg, m4a, flac) в папку `audio/` в корне репозитория.
3. Запустите сервер:

```bash
npm start
```

По умолчанию сервер поднимется на `http://localhost:3000` и отдаёт статические файлы из `public/` и аудио из `audio/`.

## Функции

- Автоматический поиск файлов в каталоге `audio/` и вывод отдельной кнопки «Воспроизвести» для каждого трека.
- При запуске нового трека предыдущий останавливается, либо плавно глушится через перекрытие, если задано время overlay.
- Индивидуальная регулировка громкости для каждого файла. Значения сохраняются в `localStorage` и восстанавливаются при повторном открытии страницы.
- Настройки overlay:
  - **Overlay transition time** — длительность перекрытия дорожек (в секундах).
  - **Overlay volume transition function** — выбор кривой изменения громкости (linear, ease-in, ease-out, ease-in-out). Настройки сохраняются в `localStorage`.

## Структура

- `server.js` — минимальный сервер на Node.js для отдачи статики и списка аудио.
- `public/` — фронтенд (HTML/CSS/JS).
- `audio/` — каталог с аудиофайлами, которые будут отображаться в интерфейсе.

## Примечания

- Приложение использует нативный `Audio` API в браузере, поэтому никаких внешних зависимостей ставить не нужно.
- Чтобы обновить список аудио после добавления файлов, просто перезагрузите страницу.
