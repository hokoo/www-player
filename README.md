# WWW Player

Мини-приложение для браузера (чистый HTML/JS/CSS) с отдельными кнопками воспроизведения для каждого аудиофайла, индивидуальными уровнями громкости и перекрытием дорожек (overlay).

## Запуск

1. Откройте файл `public/index.html` в браузере (можно двойным кликом по файлу, никакой сервер не нужен).
2. Вверху страницы нажмите «Выберите папку или файлы», укажите каталог `audio/` (или любую другую папку/набор файлов). Браузер даст странице доступ только к выбранным файлам.

## Функции

- Отображение всех выбранных аудиофайлов (mp3, wav, ogg, m4a, flac) и отдельные кнопки «Воспроизвести».
- При запуске нового трека предыдущий останавливается, либо плавно глушится через перекрытие, если задано время overlay.
- Индивидуальная регулировка громкости для каждого файла. Значения сохраняются в `localStorage` и восстанавливаются при повторном открытии страницы.
- Настройки overlay:
  - **Overlay transition time** — длительность перекрытия дорожек (в секундах).
  - **Overlay volume transition function** — выбор кривой изменения громкости (linear, ease-in, ease-out, ease-in-out). Настройки сохраняются в `localStorage`.

## Структура

- `public/index.html` — страница с интерфейсом плеера.
- `public/script.js` — логика выбора файлов, воспроизведения, громкости и перекрытия.
- `public/styles.css` — стилизация интерфейса.
- `audio/` — примерная папка для ваших треков (необязательна, используйте любую).

## Примечания

- Приложение использует нативный `Audio` API в браузере, поэтому никаких внешних зависимостей ставить не нужно.
- Чтобы обновить список аудио после добавления файлов, выберите папку/файлы ещё раз (разрешение на доступ выдаётся браузером каждый раз).
